# MAPS 2.0 — Web Standard (Next.js)

Template técnico estandarizado para landing pages optimizadas para:
- SEO local
- Motores de búsqueda
- Indexación por IA
- Escalabilidad futura
- Deploy profesional en Vercel

Este repo estandariza **estructura**, no contenido.
